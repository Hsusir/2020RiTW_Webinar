# 2020RiTW_Webinar
由中山大學管院商業大數據中心舉辦的【R愛台灣 #RiTW2020 線上研討會】，以YouTube線上直播的方式，介紹RiTW資料集內容和使用方式、R的圖形資料框(sf套件)，示範資料、圖資欄位的運算方法，及透過簡單的敘事統計搭配資料視覺化的套件，以互動網頁的方式來呈現分析的結果。
